SHELTER MANAGEMENT SYSTEM - SIMPLE VERSION
Installation Guide
====================================================================

STEP 1: SETUP WAMP
------------------
1. Make sure WAMP is installed and running (green icon)
2. Extract the shelter-simple folder
3. Copy it to: C:\wamp64\www\shelter
   (Important: Rename "shelter-simple" to just "shelter")

STEP 2: CREATE DATABASE
------------------------
1. Open browser: http://localhost/phpmyadmin
2. Click "SQL" tab
3. Copy and paste ALL content from: sql/schema.sql
4. Click "Go"
5. Copy and paste ALL content from: sql/seed_data.sql
6. Click "Go"

STEP 3: ACCESS THE SYSTEM
--------------------------
1. Open browser: http://localhost/shelter
2. Click "Login"
3. Use these credentials:

FILE STRUCTURE:

│   └── functions.php        - Database functions
├── pages/
│   ├── login.php            - Login page
│   ├── views.php            - 10 database views
│   ├── manage.php           - CRUD operations
│   └── logout.php           - Logout
├── assets/css/style.css     - Simple CSS
├── sql/
│   ├── schema.sql           - Database structure
│   └── seed_data.sql        - Sample data
└── index.php                - Homepage

TROUBLESHOOTING:
----------------
- If login doesn't work, make sure you ran BOTH SQL files
- Passwords are plain text (pw1, pw2, pw3, pw4, etc.)
- Make sure folder is named "shelter" not "shelter-simple"
